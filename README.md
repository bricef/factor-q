# factor-q

## About

`factor-q` is a personalised AI workbench on the command line.

Inspired by Claude Code and OpenCode, it takes the concept of a terminal based AI Agent beyond the code and focuses on designing, writing and planning, in the terminal.

## Features

- Design agent systems using a graph based system
- Rapid iteration over personas, skills and prompts
- Live Agent tree visualisation and customisation
- Rapid Agent definition
- Improved persistent session management and session tree visualisation
- Active learning of user habits and preferences
- Custom writing and analysis skills
- MCP framework and easy MCP integration
- Customisable and open source versioned skill library and catalogue
- Productivity integrations and tooling out of the box
- Focus on document authoring experience on the command line

## OpenAgent

A dedicated, model agnostic agent framework allowing rapid prototyping and creation of agent based systems.
